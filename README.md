# Neuro Vision

**Neuro Vision** is a fictional AI-powered cybersecurity landing page created to showcase my front-end development skills. This project demonstrates modern web design, Google Fonts integration, interactive UI elements, and responsive layouts.

---

## 🚀 Features

- **Google Fonts Integration:** Uses web fonts for branding and style.
- **Animated Hero Section:** Interactive digital eye and animated constellation background.
- **Threat Detector:** Simulated real-time threat status. (In theory)
- **Smooth Navigation:** Responsive navbar with smooth scrolling.
- **Services Overview:** Highlights AI-powered security features.
- **About & Animated Stats:** Company background with animated statistics.
- **Contact Form:** Simple form for demo purposes.
- **Responsive Design:** Looks great on desktop and mobile.

## 🛠️ Setup & Usage

1. **Clone or Download the Repository**

2. **Open Locally**
   - For best results, use a local server (e.g., VS Code Live Server extension).
   - Open `index.html` in your browser.

3. **Customize**
   - Update text, images, and contact details as needed in `index.html`.
   - Adjust styles in `styles.css` to match your branding.
